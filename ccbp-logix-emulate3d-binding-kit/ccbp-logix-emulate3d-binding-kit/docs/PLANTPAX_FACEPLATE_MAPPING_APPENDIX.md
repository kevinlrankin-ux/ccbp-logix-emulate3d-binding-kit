# PLANTPAX_FACEPLATE_MAPPING_APPENDIX.md

## Purpose
Optional guidance for mapping CCBP device structures to PlantPAx faceplates.

## Examples
- Motor Start: MTR_101.Cmd.Start → PlantPAx P_Start
- Motor Running: MTR_101.Sts.Running → PlantPAx Sts_Running
- VFD Speed Ref: AO_VFD_201_SPEED_REF_HZ → PlantPAx P_SpeedRef

Maintain CCBP tags as the source of truth.
